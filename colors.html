Colors

D8A runs monochrome with one weapon. Black for authority. Gray for hierarchy. White for space. Ship Green for the point — the color of a passing pipeline, used exactly once per surface.

Ship Green — the accent

Ship Green is CI green: the color your pipeline turns when the tests pass and the data ships. It says one thing — this is the thing that shipped. It marks the CTA, the key metric, the hero chart series, the active state, the signal pulse. Never decoration.

Gradients & glow

Four gradient recipes, and only these four. Every gradient stays inside one hue family — gray or green — or fades to transparent. Never two hues, never a rainbow.

Signal Glow
radial · green 16% → 0%
Ambient light behind hero content on dark. Atmosphere, not a statement
Canvas Depth
#0a0a0a → #0d1a13
Default dark canvas — black warming into Ink Green
Green Ramp
#00e676 → #00a85c
CTA fills and hero elements — green with depth, within the family
Green Fade
green 25% → transparent
Area-chart fills, pulse tails, section transitions
  • One family per gradient. Gray→gray, green→green, or green→transparent. A gradient is depth, not a second color.
  • Glow is ambient light, not an effect. Radial, under 20% peak opacity, behind content — never a drop shadow, never a colored border glow on cards.
  • Display headlines on dark may run White → Mint (see Fonts). Body text never gradients.
  • Still banned: gloss, bevels, multi-hue gradients, gradients on light canvas backgrounds (light stays clean — depth is a dark-canvas tool).

The one-weapon rule

  • One green statement per surface. A slide, a section, a social image, a chart — each gets exactly one green element. The CTA, or the number, or the hero series. Never several.
  • Statement vs. atmosphere. The rule counts statements — CTAs, metrics, hero series, the pulse. Atmosphere — the Signal Glow, bridge-tone surfaces, fades under 25% opacity — doesn't count. Atmosphere sets the stage; the statement makes the point.
  • Green marks the point, not the theme. If you can remove the green and the message survives, it was in the wrong place.
  • If everything is green, nothing is. When in doubt, take green away — the monochrome system works without it.

Don't

Dark-led dual system

The brand leads on black — but not flat black. Marketing surfaces sit on Canvas Depth (#0a0a0a warming into Ink Green), with the Signal Glow as ambient light behind hero content. The light canvas is for reading: docs, email, long-form — and stays clean, no gradients. Same system, same rules, different room.

Color pairing rules

  • Text on light backgrounds: Use Black (#0a0a0a) or Gray 600 (#525252) for body. Never lighter than Gray 500.
  • Text on dark backgrounds: Use White (#fafafa) or Gray 300 (#d4d4d4) for body. Never darker than Gray 400.
  • Borders: Gray 200 on light surfaces, Gray 800 on dark surfaces.
  • Emphasis: Hierarchy comes from weight, size, and contrast. Green is reserved for the single point of the surface — not for general emphasis.
  • Green on dark: Ship Green (#00e676) only. It's tuned for the black canvas.
  • Green on light: Green 600 (#00a85c) for fills and large text, Green 700 (#007a47) for body-size text and links. Ship Green itself never carries text on white — it fails contrast (1.6:1).
  • Never green-on-green, green text on mid-grays, or green paired with any other hue.

Accessibility

  • Black on White: 19.3:1 contrast ratio — WCAG AAA
  • Gray 600 on White: 7.1:1 contrast ratio — WCAG AAA
  • Gray 500 on White: 4.6:1 contrast ratio — WCAG AA (large text only)
  • White on Gray 950: 16.2:1 contrast ratio — WCAG AAA
  • Ship Green on Black: 11.8:1 contrast ratio — WCAG AAA
  • Green 600 on White: 3.0:1 contrast ratio — UI components and large text only
  • Green 700 on White: 5.2:1 contrast ratio — WCAG AA body text
  • Ship Green on White: 1.6:1 — fails. Never set #00e676 text on light backgrounds.

Why Ship Green

v1.0 reserved this slot on purpose: the monochrome system was designed to accommodate a single accent without disruption. v2.0 fills it. Green is the one color that means what D8A does — tests pass, pipeline green, shipped. And it's open territory: dbt owns orange, Databricks red, Snowflake and Azure blue, Fabric teal, Power BI yellow. Those logos sit on our own integrations row — Ship Green collides with none of them.
