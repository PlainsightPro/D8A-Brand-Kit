Colors

D8A runs light. Pearl for space, ink for authority, and one prism — the Cobalt gradient — for the point. The accent isn’t a colour; it’s light refracted. Used once per surface.

The prism — the accent

D8A is platform-agnostic: one source of truth, shown from every angle. The accent says exactly that — a single indigo refracting into blue, a spectrum from one source. It marks the CTA, the key metric, the hero chart series, the wordmark’s 8. Never decoration.

The one-weapon rule

  • One prism statement per surface. A slide, a section, a social image, a chart — each gets exactly one gradient element. The CTA, or the metric, or the hero series. Never several.
  • Statement vs. atmosphere. The rule counts statements — CTAs, metrics, hero series, the gradient headline word. Atmosphere — the Signal Mesh, tint fills, fades under 30% — doesn’t count. Atmosphere sets the light; the statement makes the point.
  • The gradient is the point, not the theme. If you can flatten the gradient to grey and the message survives, it was in the wrong place.
  • Everything else is ink on pearl. When in doubt, take the prism away — the light neutral system reads clean and premium without it.

Don't
Gradient headline, gradient icons, gradient borders, gradient background

Four gradients, zero emphasis. The prism only works refracted once.

Light-led dual system

The brand leads on light. Marketing surfaces sit on Pearl with the Signal Mesh as ambient prism light. Ink (#15121f) is the secondary canvas — product screenshots, code, the occasional bold section — where the gradient glows brightest. Same system, same rules, different room.

Accessibility

  • Ink on Pearl: 16.4:1 — WCAG AAA
  • Slate on Pearl: 8.1:1 — WCAG AAA body text
  • Mauve on Pearl: 4.9:1 — WCAG AA (muted text, captions)
  • Cobalt 600 on white: 6.5:1 — WCAG AA body text and links
  • Cobalt 700 on white: 8.5:1 — WCAG AAA
  • Blue 500 on white: 3.3:1 — large text and UI only, never body
  • Flash on Ink: 7.6:1 — the accent for eyebrows on dark surfaces
  • Prism Text is a gradient — use at 24px+ where the darkest stop still clears AA-large; never for body copy.

Why Prism

The brand explored a flat accent first — and a flat accent, in any hue, was the wrong shape for what D8A is. D8A is platform-agnostic: one source of truth, refracted across every stack you run. Iridescence is that idea made visible — a single surface throwing a spectrum, one input read from every angle. So the accent isn’t a colour, it’s a prism: a cobalt-to-blue gradient, disciplined to one statement per surface, on a clean pearl canvas. The restraint is the class; the prism is the fresh.
