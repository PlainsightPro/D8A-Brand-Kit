Common rules

  • Never use more than two weights on a single surface (e.g. Regular + Bold, or Medium + ExtraBold).
  • Headings are always Inter 700 or 800. Never use lighter weights for headings.
  • Body text line-height is always 1.6–1.7. Headings are 1.1–1.3.
  • Negative letter-spacing on headings only (28px and above). Body text uses default tracking.

CSS font stacks

--font-body: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif; --font-mono: 'JetBrains Mono', 'Fira Code', 'Consolas', monospace;

Where to get the fonts
