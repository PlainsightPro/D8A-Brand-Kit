Iconography

D8A uses Lucide icons — a clean, consistent, open-source icon set with 1px stroke weight. No filled icons, no custom icon libraries.

Icon family

We use Lucide (lucide.dev) exclusively. Lucide is the maintained fork of Feather Icons with a larger set and active development. Every icon in the D8A product comes from this family.

Icon style rules

  • Stroke only. Never use filled icon variants. D8A icons are always outline-style with 1.5–2px stroke.
  • Size: 16px, 20px, or 24px. These are the three allowed sizes. 16px for inline/compact, 20px for standard UI, 24px for prominent/nav.
  • Color: inherit from text. Icons use currentColor and match the text they sit next to. Don't color icons independently. One exception: a success or active-state icon may carry Ship Green (Green 600 on light) — and it counts as the view's one green statement.
  • No icon backgrounds. Don't put icons inside colored circles, squares, or badges. The icon stands alone.
  • Alignment: Icons are vertically centered with adjacent text. Use align-items: center in flex layouts.
  • Spacing: 8px between an icon and its label. 4px for compact layouts.

Don't

Do
Use Lucide icons at consistent sizes with stroke style, matching text color
Don't
Mix icon families (Lucide + FontAwesome + Material), use filled variants, or put icons in colored circles
